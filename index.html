About Me

My name is Hnin Haymar. I've been a backend database developer for the past 7+ years and before that I was a front-end developer, utilizing inhouse tools. I signed up for this bootcamp course because I wanted to become a full-stack developer. This whole html/css exercises have been brining back some good and painful memories of an application developer. But I really want to learn the latest and coolest frameworks like React, Angular, Backbone and Node and build interactive web applications for a couple of projects ideas I have. I am hoping through the learning and experiences of this bootcamp program, I will be able to learn all the full-stack technologies in the most efficient way possible and have the necessary tools to build them.
